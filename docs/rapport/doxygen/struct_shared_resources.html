Description détaillée

Structure regroupant les ressources partagées du serveur.

Cette structure centralise les différents éléments partagés entre les processus du serveur, permettant de gérer la synchronisation et la communication inter-processus.

Tableau des baguettes.

Ce tableau contient les structures Chopstick représentant les baguettes utilisées par les philosophes. La taille maximale du tableau est définie par la constante MAX_PHILOSOPHERS.

◆ logsQueueId

int SharedResources::logsQueueId

Identifiant de la file de logs.

◆ maxAllowedEating

sem_t SharedResources::maxAllowedEating

Sémaphore limitant le nombre maximum de philosophes pouvant manger simultanément.

Ce sémaphore est initialisé à la valeur du plancher du nombre total de philosophes divisé par 2, garantissant qu'au maximum la moitié des philosophes puissent manger en même temps.

◆ philosopherCreationProcess

sem_t SharedResources::philosopherCreationProcess

Sémaphore permettant de sécuriser la création concurrente de philosophes.

Ce sémaphore assure que l'accès à la mémoire partagée lors de la création de nouveaux philosophes est correctement synchronisé lorsque plusieurs processus tentent de créer un philosophe simultanément.

Tableau des philosophes côté serveur.

Ce tableau contient les structures ServerPhilosopher représentant chaque philosophe. La taille maximale du tableau est définie par la constante MAX_PHILOSOPHERS.
