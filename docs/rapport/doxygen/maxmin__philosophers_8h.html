Description détaillée

Définit les macros de configuration pour le nombre de philosophes et les temps d'état.

Ce fichier d'en-tête définit plusieurs constantes utilisées pour paramétrer l'application relative au problème des philosophes.

Les macros définies sont :

  • MIN_PHILOSOPHERS : Le nombre minimum de philosophes requis pour démarrer le système (2).
  • MAX_PHILOSOPHERS : Le nombre maximum de philosophes autorisés dans le système. Cette valeur est arbitraire mais doit être inférieure ou égale à la macro FOPEN_MAX.
  • MIN_STATE_TIME : Le temps minimum (en secondes) qu'un philosophe doit passer dans un état donné.
  • MAX_STATE_TIME : Le temps maximum (en secondes) qu'un philosophe peut passer dans un état donné.
Note
Ces valeurs permettent de contrôler le comportement de l'application et d'assurer que le système dispose d'un nombre adéquat de philosophes pour fonctionner correctement.
